### Usage



* **环境**

```shell
$ docker pull mongo

$ cd Database

$ docker-compose up

$ pip3 install -r requirments.txt
```



**启动docker-compose后运行**

* **运行**

```shell
$ python3 server.py
```



* **测试**

```shell
$ python3 -m unittest
```
