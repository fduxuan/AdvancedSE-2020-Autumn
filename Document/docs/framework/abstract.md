## 2. 贡献

在重构过程中，我们实现了以下贡献：



* 使用python flask重构了整个后端
* 对原本的H2内存数据库做了优化，改成了真实存在的mongo型数据库
* 采用了全新的consul服务作为注册中心进行服务注册和发现
* 实现服务间通信
* 使用RabbitMq-Python 构建消息队列进行异步通信，与前端采用websocket相连接
